# Admin Dashboard Pro

Professional admin dashboard built with `Next.js`, `Neon` and a custom API for authentication and user management.

## Stack

- `Next.js` with App Router
- `React 19`
- `Tailwind CSS`
- `Recharts`
- `Neon Serverless Postgres`
- Custom API routes in `app/api`

## Features

- Admin login with secure `httpOnly` cookie session
- Protected dashboard, users and settings pages
- User CRUD backed by Postgres
- Dashboard charts by city and company
- Dark mode preference on the client
